**版本**：V1.0  
**适用**：手工功能测试、回归测试、发版前验收

## 十六、端到端测试场景（E2E）

以下为完整用户旅程，需按顺序手工执行并勾选验证点。

---

### E2E-01 新用户完整流程（P0）

**目标**：新用户从打开小程序到完成首次验收的完整路径。

| 步骤 | 操作 | 预期结果 | 执行结果 |
|------|------|----------|----------|
| 1 | 清除缓存后打开小程序 | 展示引导页 | |
| 2 | 点击「开始使用」 | 进入首页 | |
| 3 | 点击城市入口，选择城市（如深圳）并确认 | 城市保存，首页显示城市简称 | |
| 4 | 点击「装修公司检测」 | 进入公司检测页 | |
| 5 | 输入公司名，点击「开始检测」 | 进入检测中页 | |
| 6 | 等待检测完成 | 自动或手动进入公司风险报告详情 | |
| 7 | 返回首页，点击「AI 施工验收」 | 进入施工陪伴页 | |
| 8 | 设置开工日期（如明天）并确认 | 展示 S00～S05 阶段卡片 | |
| 9 | 点击 S00「人工核对」 | 进入材料核对页 | |
| 10 | 为至少一项材料上传照片，点击「核对通过」 | 提交成功，返回施工页，S00 已核对、S01 可点 | |
| 11 | 点击 S01「AI 验收」 | 进入验收/拍照页 | |
| 12 | 上传至少 1 张验收照片并提交分析 | 进入分析中 | |
| 13 | 等待分析完成 | 进入验收结果页，有通过或待整改 | |
| 14 | 返回施工陪伴页 | S01 状态已更新，S02 已解锁（若 S01 通过） | |

**验证点**：引导→首页→城市→公司检测→施工→开工日期→S00 材料核对→S01 验收→状态与互锁均正确。

### E2E-02 施工陪伴 6 阶段全流程（P0）

**目标**：已设置开工日期用户，按顺序完成 S00～S05 全流程。

| 步骤 | 操作 | 预期结果 | 执行结果 |
|------|------|----------|----------|
| 1 | 进入施工陪伴页，确认已有开工日期和 S00～S05 | 6 阶段展示，S00 可操作，S01～S05 根据当前状态解锁或置灰 | |
| 2 | 完成 S00 材料核对（上传照片并核对通过） | S00 已核对，S01 解锁 | |
| 3 | 完成 S01 隐蔽工程验收（上传照片、分析通过） | S01 已通过，S02 解锁 | |
| 4 | 依次完成 S02～S05 验收 | 每阶段通过后下一阶段解锁，最终 S05 已通过 | |
| 5 | 进入「我的」→「数据管理」 | 可看到施工照片、验收报告等数据 | |

**验证点**：流程互锁正确，阶段状态与数据管理一致。
