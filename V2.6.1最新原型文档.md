# 装修决策Agent微信小程序完整原型文档（V15.3 → V2.6.8优化版）
**适配场景**：直接交付Cursor/开发团队，可无缝解析开发，含36个页面全维度细节（定位/布局/交互/数据/联动/视觉规范）
**核心能力**：6大装修阶段流程互锁+**阶段开始/验收前3天微信+小程序双重智能提醒**，AI核对/验收/报告生成/台账管理全闭环
**技术基准**：微信小程序原生规范，基准尺寸375×667，rpx弹性单位，适配全机型/深色模式，所有交互带动效/兜底/权限校验逻辑
**核心规则**：前置阶段未通过则后续阶段强制锁定，提醒功能支持全局开关/自定义天数，所有操作后端校验+前端屏蔽，无绕过可能

## V2.6.5 复检次数与解锁规则优化（2026-02）

### 1. 解锁规则（同阶段不重复收费）
- **同一阶段验收报告**：解锁一次即可，复检不重复收费
- 解锁后：初次验收 + 所有复检均在权益内

### 2. 复检次数限制（最多3次）
- **每阶段最多申请复检 3 次**
- 展示「还可申请复检 X/3 次」
- 第 3 次提交前弹窗提示「本次为最后一次复检机会」
- 3 次后：隐藏「申请复检」按钮，展示「复检次数已用完」，引导咨询 AI 监理或进入下一阶段

### 3. 3 次后自动进入下一阶段
- 复检 3 次仍未通过：阶段状态置为 `rectify_exhausted`
- **施工陪伴页**：该阶段显示「待整改」，但下一阶段可解锁
- 后端 `_stage_passed` 将 `rectify_exhausted` 视为可进入下一阶段

### 4. 状态映射
- `rectify` / `need_rectify` / `pending_recheck` → 待整改（下一阶段锁定）
- `rectify_exhausted` → 待整改（下一阶段可解锁）

### 2. 智能提醒全局规则（三重触达/微信规范适配）
1.  **触达方式**：微信服务通知（点击直达对应阶段）+ 小程序内消息中心（P14，未读角标）+ P02/P09页面红点（8rpx纯红圆形，#FF3333，进入对应页自动消除）
2.  **权限规则**：首次进入P09弹出微信「长期订阅/一次性订阅」授权弹窗，拒绝后仅保留小程序内提醒，P19支持重新授权/开关
3.  **自定义规则**：提醒提前天数支持1/2/3/5/7天选择，全局统一生效，P09/P19双向实时同步
4.  **联动规则**：前置阶段未通过→后续阶段提醒自动暂停；阶段状态=已完成→该阶段提醒永久失效；手动校准阶段时间→提醒时间同步刷新
5.  **推送时机**：精准触发「阶段开始前N天」「阶段验收前N天」，后端每分钟扫描任务表，无延迟/重复推送

### 3. 视觉/交互通用规范（所有页面统一）
#### 视觉规范
- **色值**：主色`#007AFF`、成功色`#00CC66`、警示色`#FF9900`、危险色`#FF3333`、中性色`#333333`/`#666666`/`#999999`/`#EEEEEE`
- **字体**：标题≥32rpx（加粗）、正文28rpx、辅助文字24rpx、最小提示文字20rpx（不低于此尺寸）
- **按钮**：主按钮88rpx高、圆角16rpx、线性渐变`#007AFF`；次要按钮60rpx高、圆角16rpx、边框1rpx`#007AFF`；最小点击区域≥44rpx×44rpx
- **卡片**：圆角16rpx、阴影`0 2rpx 8rpx rgba(0,0,0,0.1)`、间距20rpx
- **红点**：提醒红点8rpx纯红圆形（`#FF3333`），位于元素右上角10rpx处，无数字
- **弹窗**：遮罩层`rgba(0,0,0,0.7)`（深色模式0.9），弹窗宽90%、高按需，居中显示，支持空白处/右上角×关闭

#### 交互规范
- **动效**：弹窗0.2s淡入淡出、按钮点击0.1s按压缩放（scale=0.95）、页面切换0.2s左右滑动、折叠/展开0.2s平滑过渡
- **Toast**：固定在胶囊按钮下方，背景`rgba(0,0,0,0.8)`、文字`#FFFFFF`、显示3秒，期间禁止重复触发
- **加载**：微信原生加载图标（60rpx×60rpx），搭配进度条（6rpx高、圆角3rpx、主色`#007AFF`）
- **上传**：支持断点续传、进度实时更新，单文件≤10MB，仅支持PDF/JPG/PNG
- **拍照**：强制竖拍，支持AI辅助裁剪、相册选择，最多9张，关联阶段专属拍摄指引
- **流程互锁**：未满足前置条件→对应按钮置灰（`#E5E5E5`）+ 不可点击 + Toast提示前置要求，无绕过可能

### 4. 数据/跳转通用规则
- **页面跳转**：所有跳转携带**用户ID + 阶段编号（如有）+ 操作参数**，示例：P02→P09?uid=xxx&stage=S01
- **数据同步**：所有前端操作实时同步后端，后端返回状态码（200=成功/400=参数错误/403=权限不足/409=流程互锁），前端根据状态码做对应反馈
- **本地缓存**：仅缓存非敏感信息（如引导页完成标记、页面红点状态），敏感信息（如开工日期、阶段状态）仅从后端拉取
- **兜底处理**：网络异常跳转P33、空数据显示专属空视图、操作超时显示「点击重试」

   - 再次点击其他热门城市→自动切换选中状态，原选中城市恢复默认样式。
3. **省份-城市选择**：
   - 点击左侧省份→右侧城市列表实时刷新，显示该省份下所有城市，选中省份自动置顶；
   - 点击右侧城市→城市文字变为`#007AFF`+背景高亮，底部确认按钮高亮；
   - 支持省份/城市列表上下滑动，滑动时不改变当前选中状态。
4. **城市搜索**：
   - 输入城市名/拼音→实时筛选匹配结果（如输入"sz"显示"深圳"），搜索结果显示在搜索框下方，点击可直接选中；
   - 清空搜索输入→自动恢复显示热门城市+省份-城市选择区。
5. **确认选择**：
   - 点击「确认选择」→保存选中城市至本地缓存，返回上一页；
   - 切换城市后弹出Toast提示："您已切换至[城市名]，后续AI分析将基于该城市的本地规范，历史报告不受影响"（24rpx`#FFFFFF`，背景`#333333`，显示3秒）；
   - 网络异常时→弹出提示"城市保存失败，请检查网络后重试"（24rpx`#FF3333`），保留原选中城市，不返回上一页。
6. **动效**：
   - 省份/城市列表滑动带惯性动效，选中状态切换带0.1s背景渐变；
   - 搜索结果弹出带0.2s淡入动效，确认按钮点击带0.1s按压缩放。

#### 异常场景处理
| 异常类型   | 触发条件              | 页面反馈                                       |
| ---------- | --------------------- | ---------------------------------------------- |

| 网络异常   | 确认选择时网络断开    | 弹出失败提示，保留当前选中状态，不跳转页面     |
